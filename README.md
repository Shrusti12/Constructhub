# ConstructHub

## 📌 Overview

ConstructHub is a web-based construction project management platform designed to simplify project tracking, team collaboration, and task management. The application provides an intuitive interface for managing construction-related activities and monitoring project progress efficiently.

## 🚀 Features

* User Authentication and Authorization
* Project Creation and Management
* Task Assignment and Tracking
* Interactive Dashboard
* Responsive User Interface
* REST API Integration
* Database Management with PostgreSQL
* Secure Data Handling

## ⚙️ Installation

### Clone the Repository

git clone https://github.com/Shrusti12/ConstructHub.git

### Frontend Setup

cd frontend

npm install

npm run dev

### Backend Setup

cd backend

pip install -r requirements.txt

python app.py

## 🗄️ Database Configuration

Configure PostgreSQL credentials in the environment file before running the application.

Example:

DB_HOST=localhost

DB_NAME=constructhub

DB_USER=postgres

DB_PASSWORD=your_password
